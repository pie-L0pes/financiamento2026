# 📱 Simulador de Financiamento

## 📌 Descrição do projeto

Este projeto é um aplicativo mobile desenvolvido em Dart utilizando o Flutter que simula um financiamento com base em juros simples.

O usuário pode inserir:

* Valor do financiamento
* Taxa de juros mensal
* Número de parcelas
* Taxas adicionais

A partir desses dados, o aplicativo calcula:

* Valor total a ser pago
* Valor de cada parcela

O objetivo é auxiliar na visualização dos custos antes de realizar uma compra parcelada.
